## 搭建Github图床

### 1. 创建Token

- 登录Github，点击右上角头像，选择Settings
- 选择Developer settings
- 选择Personal access tokens
- 点击右上角Generate new token
- 设置Token名称，选择Token有效期，勾选repo
- 点击Generate token，生成Token
